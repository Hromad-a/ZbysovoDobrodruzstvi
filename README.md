# Zbýšovo dobrodružství

Cozy top-down walking game — walk a black cat on a leash through a Prague-style
panelák neighbourhood, meet philosophizing hedgehogs, reach the person waiting
on the other side.

Sketchy child-drawing / Die Brücke placeholder art generated with
[Rough.js](https://roughjs.com) (boiling-line animation, hand-drawn PNGs to be
swapped in later).

## Run

It's a single `index.html` — open it in a browser, or serve it:

```
python3 -m http.server 8000
```

Works on desktop (WASD / arrows) and mobile (floating touch joystick).

## Status

Playable placeholder prototype: character select, leash physics, cat AI
(wander / sniff / hedgehog encounters), 8 hedgehogs with Czech dialogue,
ending sequence with petting.
